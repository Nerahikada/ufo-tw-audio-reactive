# ufo-tw-audio-reactive

音声ファイルを解析し、BLE 経由で U.F.O. TW デバイスのモーターをリアクティブに制御する Web アプリ。

## 概要

音声の L/R チャンネルをリアルタイムに解析し、ステレオ分離に基づいて 2 つのモーター（Parent / Child）の速度と回転方向を独立制御する。ビート検出による方向転換、余韻（サステイン）、LR 分離度の調整など、多彩なパラメーターで動きをカスタマイズできる。

## 使い方

ビルド不要。任意の HTTP サーバーで静的ファイルを配信するだけで動作する。
Web Bluetooth API を使うため、`localhost` または HTTPS 環境が必要。

1. ブラウザで開く
2. 「BLE 接続」で U.F.O. TW デバイスにペアリング
3. 音声ファイルを選択して再生

## ファイル構成

| ファイル | 内容 |
|---|---|
| `index.html` | UI |
| `style.css` | スタイル |
| `app.js` | メインロジック（UI イベント、モーター制御ループ） |
| `audio-reactive.js` | 音声解析（RMS、FFT、ビート検出、ステレオ分離） |
| `audio-player.js` | 音声プレイヤー（ファイル読み込み、再生制御） |

| `testing/` | BLE プロトコル調査・API テスト用ツール |

## パラメーター

- **感度** — 音量に対するモーター反応の強さ
- **ノイズゲート** — この閾値以下の音量ではモーター停止
- **スムージング** — レベル変化の平滑化
- **余韻** — 音が止まった後もモーターが回り続ける度合い
- **LR分離度** — 0（モノラル）〜 1（完全ステレオ）
- **方向転換の間隔** — ビート検出による回転方向切り替えのクールダウン
